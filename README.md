# search-service
Simple search service use Naver api.

## Install
### pip install -r requirements.txt
---
## Environment
### add .env file
```
FLASK_APP=app
FLASK_ENV=development
FLASK_DEBUG=1
```
---
## Endpoints
### Root
URL
```
POST {domain}/
```
BODY
```
Hello, search-service
```
